# PDF生成テスト手順

$ARGUMENTS

## 1. PDF生成実行
```bash
php run.php
```

## 2. 確認項目

### 個別PDF（temp/フォルダ）
- 各テンプレートタイプが生成されているか
- 画像が正しく配置されているか
- テキストが枠内に収まっているか

### 印刷用PDF（draft/フォルダ）
- ファイルサイズが適切か（極端に小さい/大きい場合は問題あり）
- 回転が正しく適用されているか（ROTATE=1の場合）
- DRAFTS_XYS座標で正しく配置されているか

## 3. テンプレートタイプ別チェック
- `_text`: テキスト要素のみ
- `_fullcolor`: フルカラー画像
- `_futa`: 二度塗り用（黒埋め）

## 4. 問題発生時
- 直前の変更を確認
- 座標変更があった場合は元に戻す
- 回転処理を追加した場合はStartTransform/StopTransformを確認

## 5. ファイルサイズ目安
- 個別PDF: 数十KB〜数百KB
- 印刷用PDF: 数MB程度
- 極端に小さい場合: 画像が含まれていない可能性
- 極端に大きい場合: 画像圧縮に問題がある可能性
